# PowerBI--Gym-Dashboard
An interactive power bi dashboard which is used to analyse the gym performance and provide business insights through visualizations.
Features-
Interactive dashboard
KPI cards
Sales trend analysis
Profit analysis
Region-wise performance
Category-wise sales
Slicers and filters for dynamic reporting

Tools & Technologies-
Power BI
Power Query
DAX
Microsoft Excel / CSV

Dataset-
The dashboard uses a gym dataset for visualization and business analysis.
